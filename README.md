# dados-abertos

## Arquitetura Visulizacao de Dados

Para o projeto de visualizacao de dados abertos, sera necessario importar o CSV contendo as seguintes informacoes:

* Ano/Mês Referência
* Ano/Mês Competência
* UF
* Código Município SIAFI
* Nome Município SIAFI
* CPF Beneficiário
* NIS Beneficiário
* Nome Beneficiário
* Valor Parcela
